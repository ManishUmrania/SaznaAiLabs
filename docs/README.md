## Getting Started

If you're new to the Sazna Platform, we recommend following this learning path:

1. **Start with the Main README** - Review the main project README for an overview
2. **Understand the Architecture** - Read the architecture documentation
3. **Set up Development Environment** - Follow the local development guide
4. **Learn the APIs** - Study the API guide for integration
5. **Explore Deployment Options** - Review deployment strategies
6. **Ensure Quality** - Follow testing guidelines
